# HIRAIZUMI-YAKON
## 🦸‍♂️ YAKONMAN - 平泉腸の守護神

YAKONMANは、平泉町を舞台に腸活の大切さを伝える教育ヒーローです。  
ヤーコンの力と地域の誇りを融合し、子どもたちに腸内環境の知識を楽しく届けます。

![YAKONMAN](./assets/images/yakonman_hiraizumi_guardian.png)

### キャッチコピー
> 「平泉腸の守護神」

### 特徴
- 胸元に腸型の家紋
- ヤーコン剣と腸活スキャナーを装備
- 中尊寺金色堂を背景にしたビジュアル
